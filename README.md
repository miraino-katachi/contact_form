# contact_form
【PHP】お問い合わせフォーム

入力フォームを作成します。下記のことを練習できます。
- フォームから入力した値の受け取り
- ワンタイムトークンを使ったCSRF対策
- 受け取った値のサニタイジング（XSS対策）
- 受け取った値のバリデーション
- セッションの使い方
